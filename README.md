# SketchRumble 

SketchRumble is a real time multiplayer drawing and guessing game..One player draws a secret word while others race to guess it :)

1. Open https://collaborative-drawing-app.vercel.app/ and enter your display name  
2. It asks for you to wait the game starts only when **more than one player is present** so just open INCOGNITO and enter your another display name for test 
3. Then the round starts one have to select one of the 3 words presented on your screen to draw
4. Use various tools from the header section to draw <img width="1853" height="1022" alt="image" src="https://github.com/user-attachments/assets/e43f502e-3498-4df5-a934-8f07f29b9159" />

5. The oposition player will guess the name in the chat box while slowly the each letter will be revealed at the top
6. This goes upto 5 rounds each round runs till every participants turn completed

7. There is also the private room feature only for your friends no one else can join, you just have to share the link

**Used Tech Stack:** 

React  

Express  

Node  

WebSockets (Socket.io, most important for collaborative work) and heavy work on layout
